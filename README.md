# Pune Metro Route & Fare Planner

A **console-based Java project** to calculate the shortest route, travel time, number of interchanges, and fare between Pune Metro stations.  

This project uses **Graphs** and **Heap data structures** along with **Dijkstra's algorithm** to find optimal routes.

---

## Features

- Shortest distance between source and destination stations  
- Estimated travel time  
- Number of interchanges  
- Fare calculation based on distance  
- User-friendly console menu for selecting actions

---

## How to Run

1. Make sure you have **Java JDK installed** (version 21 recommended).  
2. Open a terminal/command prompt in the project folder.  
3. Compile the Java files:

```bash
javac Heap.java MainGraph.java
